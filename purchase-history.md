## System Design & Fitting Analysis

**Analysis date:** 2026-05-01
**Water source:** 60 PSI, ~6 GPM (5 gal / 50 sec at top of hill through 1in hose)

### Physical Layout

```
WATER SOURCE (hose bibb + backflow preventer)
       │
    1in POLY TRUNK
       │
    1in TEE ─── splits ──┐
       │                  │
       ▼                  ▼
   VALVE BOX 1        VALVE BOX 2
   (4 valves)         (5 valves)
   ┌─────────┐        ┌──────────────┐
   │ 1in manifold     │ 1in manifold  │
   │ ├─V0 → 4 rotors  │ ├─V4 → 4 rotors (Southeast)
   │ ├─V1 → 4 rotors  │ ├─V5 → 3 rotors (South)
   │ ├─V2 → 4 rotors  │ ├─V6 → 4 rotors (Southwest)
   │ └─V3 → 4 rotors  │ ├─V7 → drip (Peonies)
   └─────────┘        │ └─V8 → drip (Garden)
   Front A/B +        └──────────────┘
   Backyard A/B
   = 16 heads          = 11 heads + 2 drip
```

**Totals:** 9 valves, 27 rotor heads, 2 drip zones.

### Inside Valve Boxes

```
VB1 — 1in poly feed enters:

1in poly ─┬─ 1in TEE ─── 1in×¾in ADAPTER ─── VALVE 0 ─── ¾in ADAPTER ─── ¾in poly → 4 rotors
          ├─ 1in TEE ─── 1in×¾in ADAPTER ─── VALVE 1 ─── ¾in ADAPTER ─── ¾in poly → 4 rotors
          ├─ 1in TEE ─── 1in×¾in ADAPTER ─── VALVE 2 ─── ¾in ADAPTER ─── ¾in poly → 4 rotors
          └─ 1in PLUG── 1in×¾in ADAPTER ─── VALVE 3 ─── ¾in ADAPTER ─── ¾in poly → 4 rotors

VB2 — 1in poly feed enters:

1in poly ─┬─ 1in TEE ─── 1in×¾in ADAPTER ─── VALVE 4 ─── ¾in ADAPTER ─── ¾in poly → 4 rotors
          ├─ 1in TEE ─── 1in×¾in ADAPTER ─── VALVE 5 ─── ¾in ADAPTER ─── ¾in poly → 3 rotors
          ├─ 1in TEE ─── 1in×¾in ADAPTER ─── VALVE 6 ─── ¾in ADAPTER ─── ¾in poly → 4 rotors
          ├─ 1in TEE ─── 1in×¾in ADAPTER ─── VALVE 7 ─── NIPPLE ─── P.REG ─── drip
          └─ 1in PLUG── 1in×¾in ADAPTER ─── VALVE 8 ─── NIPPLE ─── P.REG ─── drip
```

### Connection Chain: Lateral → Rotor Head

```
¾in poly lateral
    │
¾in barb TEE (branch off to head, main run continues)
    │
¾in FNPT ADAPTER (barb in poly, FPT accepts swing elbow)
    │
½in barb × ¾in MPT SWING ELBOW (threads into FNPT)
    │
½in SWING PIPE (~12-18in)
    │
½in barb × ½in MPT SWING ELBOW (threads into rotor)
    │
RAIN BIRD 42SA+ ROTOR (½in FPT inlet)
```
